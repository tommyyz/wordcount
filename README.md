# wordcount
word count in various languages: js, go, python...
不同开发语言版本的字数统计小工具

### Quick Start
```
node wordcount.go
# [ 'This', 'is', 'Tom', 'Zhu', '这', '是', '汤', '姆', '猪' ]
# 9
```

```
go run wordcount.go
# [This is Tom Zhu 这 是 汤 姆 猪] 9
```
